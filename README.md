# All-About-Mechanisms
This repository serves as a centralized archive for my mechanical engineering design and simulation projects. It documents the transition from theoretical mechanism synthesis to physical implementation.

🛠 Project Categories

CAD Models (Onshape/SolidWorks): Structural designs for robotics, aerospace components, and mechanical assemblies.

Mechanism Synthesis: Programs and scripts used to analyze motion, kinematics, and dynamic loads.

Robotics: Design files and documentation for custom chassis and robotic arms.

💻 Tech Stack

CAD: Onshape, SolidWorks

Programming: Python (for simulation and kinematics)

Methodology: Mechanism synthesis, Finite Element Analysis (FEA) theory

📁 Repository Structure

/cad-designs: Exported parts and assembly renders.

/simulations: Python scripts for motion analysis.

/docs: Technical reports and documentation.
